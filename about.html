About TVScope

What is TVScope?

TVScope lets you search for any TV show, save it to your personal watchlist, and mark episodes as watched. All data is stored in your browser — no account needed.

How does it work?

Search results come from the free TVmaze API — no API key required. Your watchlist is saved in localStorage so it survives page refreshes.

Tech Stack

  • HTML5 — semantic structure
  • CSS3 — Flexbox, Grid, custom properties
  • JavaScript — ES modules, async/await
  • TVmaze API — free, no registration
  • localStorage — client-side persistence

API Info

Endpoint used: https://api.tvmaze.com/search/shows?q=query

Free tier, no API key, no rate limits for search.
